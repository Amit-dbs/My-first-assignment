⚙️ App Settings
📋 Setup Steps
1. Open sheets.new — create blank spreadsheet
2. Extensions → Apps Script → paste downloaded code → Save
3. Deploy → New Deployment → Web App → Anyone → copy URL
4. Paste URL above → Save & Connect
5. Telegram: message @BotFather → /newbot → copy token
6. Apps Script → Project Settings → Script Properties → add TG_TOKEN and TG_CHAT
7. Apps Script → Triggers → scheduledDailySummary → Day timer → 5 AM
